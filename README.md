# Public Transport Rental & Sharing System (PTRS)

A Laravel-based full-stack web application that allows vehicle owners (renters) to share their vehicles and customers to rent them efficiently through a secure booking system powered by RESTful API integration.

## Features
- User Authentication (Login & Register)
- Role-based Access: Renter & Customer
- Vehicle Sharing & Rental System
- Dashboard Management
- CRUD Operations
- RESTful API Integration
- Responsive UI Design (BootStrap)

## Tech Stack
- Laravel
- PHP
- MySQL
- HTML, CSS, Bootstrap
- RESTful API
